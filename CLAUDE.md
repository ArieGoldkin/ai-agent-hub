## üì¶ What This Repository Contains

This is the **source repository** for the NPM package `ai-agent-hub`:

```bash
npx ai-agent-hub@latest
```

**Repository Structure (Source Materials):**
- `/agents/` - 9 specialized agent personality definitions (source)
- `/assets/` - Installation templates and instruction files (source)
- `/skills/` - 14 expert knowledge modules
- `/lib/` - TypeScript source for CLI and generators
- `/bin/` - CLI entry point and commands
- `/.squad/` - Squad mode orchestration system
- **NO `.claude/` folder** - Created during installation, not stored here

**What Gets Published to NPM:**
- All folders above (except `.claude/` which doesn't exist)
- Installation script generates `.claude/` from `/agents/` and `/assets/`
- Users get a complete `.claude/` setup created automatically

## üéØ Development Workflow

### Source Files (Edit These)

| Location | Purpose | Published |

### Generated Files (Created by Installation)

| Location | Created When | Source |
|----------|--------------|--------|
| `.claude/agents/` | User runs install | Copied from `/agents/` |
| `.claude/instructions/` | User runs install | Copied from `/assets/instructions/` |
| `.claude/context/` | User runs install | Fresh initialization |
| `CLAUDE.md` | User runs install | Generated by `/lib/claude-md-generator/` |

### üõ†Ô∏è Common Development Tasks

1. **Adding a new agent**:
   - Create/edit in `/agents/my-agent.md`
   - Agent will be installed to user's `.claude/agents/` automatically

2. **Updating instructions**:
   - Edit `/assets/instructions/*.md`
   - Changes appear in user's `.claude/instructions/` on next install

3. **Creating a skill**:
   - Add to `/skills/my-skill/SKILL.md` with metadata
   - Include bundled resources if needed

4. **Testing installation locally**:
   ```bash
   npm run build
   node dist/bin/cli.js
   # This creates a .claude/ folder for testing (gitignored)
   ```

5. **Publishing**:
   ```bash
   npm version [patch|minor|major]
   npm publish
   ```

## üîç Architecture: Source vs. Generated

### This Repository (Source)
- **SOURCE FILES**: `/agents/`, `/assets/`, `/skills/`

- Full TypeScript source in `/lib/`

- **GENERATED**: `.claude/` folder created fresh
- Agents copied from `/agents/` source
